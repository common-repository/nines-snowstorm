== Description ==
本插件 是一种 JavaScript 驱动的雪景效果，可以很容易地添加到网页中。它是免费使用的，并且易于设置。

更多插件:https://www.aliluv.cn



== Installation ==

1. 上传 `nines-snowstorm`目录 到 `/wp-content/plugins/` 目录或后台搜索 `特效`
2. 在后台插件菜单激活该插件
